Sea level predictor

This project analyzes and visualizes sea level rise trends using historical data. It employs Python libraries like NumPy, Pandas, Matplotlib, and SciPy to process data from a CSV file, create a scatter plot of sea level measurements over time, and generate two lines of best fit for different time periods. The first line projects the trend from 1880 to 2050, while the second focuses on data from 2000 onward, offering a more recent perspective. The resulting plot, saved as an image, provides a clear visual representation of past sea level changes and future projections, serving as a tool for understanding long-term environmental trends related to climate change.

Medical data visualizer

This Python script analyzes and visualizes medical examination data to explore relationships between various health factors and cardiovascular disease. It begins by importing essential data analysis and visualization libraries, then loads and preprocesses a medical dataset. The script calculates Body Mass Index (BMI) and creates new categorical variables for overweight status, cholesterol, and glucose levels. Two main visualization functions are implemented: one generates a categorical plot showing the distribution of various health factors in relation to cardiovascular disease, while the other creates a correlation heatmap of medical metrics. The categorical plot function uses data melting and grouping techniques to prepare the data, then utilizes seaborn to create a multi-panel bar plot. The heatmap function filters out outliers, calculates correlations between variables, and visualizes these relationships using a color-coded matrix. Both visualizations are saved as image files, providing valuable insights into the complex interplay of health factors and their association with cardiovascular disease risk.

Time series data visualizer

This project analyzes and visualizes daily page view data from the FreeCodeCamp forum spanning from May 2016 to December 2019. The Python script uses pandas for data manipulation and matplotlib and seaborn for creating informative visualizations. It generates three types of plots: a line plot showing daily page views over time, a bar plot displaying average monthly page views for each year, and box plots illustrating both yearly trends and monthly seasonality. The script also includes data cleaning by removing outliers. These visualizations help identify patterns, trends, and potential anomalies in the forum's page view statistics, providing insights into user engagement over the years.

European bike sales analysis

In this data analysis project, I processed over 1,000 records of European bike sales data using Microsoft Excel. After performing thorough data cleaning within the spreadsheet, I leveraged Excel's pivot table functionality to organize and summarize the information effectively. The key focus was on examining the relationship between bike purchases and customer income, segmented by gender. Utilizing Excel's charting capabilities, I created a stacked bar chart that clearly illustrates the average income of bike purchasers versus non-purchasers, with distinct groupings for male and female customers. This Excel-based visualization enables quick identification of income-related patterns in bike-buying behavior across genders, providing valuable insights for targeted marketing strategies and product positioning.
